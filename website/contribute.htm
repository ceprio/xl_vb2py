Contributions are always welcome and there are many ways that you can get involved in the project.

· Offering to test the software on your VB projects

· Submitting VB code fragments for testing

· Submitting Python code to add functionality

· Offering advice!

The best ways to start are to post to the forums, email paulpaterson@users.sourceforge.net or to add to the Wiki.

To keep up to date with the latest project news or to ask direct questions, you can use the mailing list or the Forums.
