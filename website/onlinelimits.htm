The disclaimer

1. Your code is submitted to the server as plain text. Please do not submit commercially sensitive code.
2. Your code is never stored on the server unless you explicitely give your permission.
3. If you are not comfortable with either of these constraints then please just download the latest version.

Limitations

1. The online version only handles one file at a time. Full projects should be coming later.
2. Because it doesn't handle projects you cannot handle forms.
3. Conversion is to Python 2 only for the moment.
