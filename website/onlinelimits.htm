First the disclaimer (you have been warned!)

1. Your code is submitted to the cgi script as plain text. This is insecure. Please do not submit commercially sensitive code.
2. Your code is never deliberately stored on the server although it may end up in the log files.
3. If you have a problem with either of these issues the just download the latest version - the online version is not intended as a serious conversion environment and it isn't secure.

 

Now the limitations!

1. 500 lines of code maximum. Parsing is pretty CPU and memory intensive and we are running of the Sourceforge servers. Please be considerate!
2. There is no way to adjust the options (yet). You have to live with the defaults. If there is enough interest then I may add this.
3. You can only do a conversion in the 'context' of a Code module, Class module or Form module.
