 The vb2Py project is developing a suite of conversion tool to aid in translating existing Visual Basic projects into Python.

The conversion includes,

· VB code modules translating to Python code modules

· VB classes to Python classes

· VB Forms to PythonCard forms

· VB Projects to PythonCard projects

The project is in its early stages right now with work progressing on the following modules,

· A VB Parser

· A VB code converted

· A .frm/.frx to PythonCard resource file translator
