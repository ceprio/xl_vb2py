Tough Example - Resuming Errors

VB Code

On Error Goto HANDLER
    Statement1
IGNORE:
    Statement2
    StatementN
HANDLER:
    If Cond1 Then
        Resume
    Else If Cond2 Then
        Retry
    Else
        Resume IGNORE
    End If
Python Code


while 1:
    try:
        Statement1
    except:
        if Cond2: continue
        break

while 1:
    try:
        Statement2
        StatementN
    except:
        if Cond1: break
